# AHAD-IBM-BOB-
# AI Hyper-Local Supply Predictor

કોઈપણ કોડિંગ જ્ઞાન વગર સ્થાનિક દુકાનદારો માટે હવામાન, તહેવારો અને ઇન્વેન્ટરીના આધારે સપ્લાયનું અનુમાન લગાવતી એપ્લિકેશન.

## આર્કિટેક્ચર (How It Works)
1. **ડેટા સોર્સ:** હવામાન API અને BOB ડેટાબેઝમાંથી કરંટ સ્ટોક આપોઆપ ખેંચાય છે.
2. **AI પ્રોસેસિંગ:** ડેટા IBM watsonx.ai (AutoAI Time-Series) મોડલમાં સબમિટ થાય છે.
3. **ઓટોમેશન (BOB):** 
   - દર શનિવારે સવારે દુકાનદારને વોટ્સએપ પર સપ્લાય ટ્રેન્ડનું એલર્ટ મોકલે છે.
   - જો સ્ટોક ઓછો હોય, તો હોલસેલરને ઓટોમેટીક ઓર્ડર ઇમેઇલ ડ્રાફ્ટ કરે છે.

## ઈન્સ્ટોલેશન ગાઈડ (No-Code Execution)
1. આ રિપોઝીટરીમાંથી `flows.json` કોડ કોપી કરો.
2. તમારા Node-RED અથવા BOB બેકએન્ડ ડેશબોર્ડમાં જાઓ.
3. **Import** ઓપ્શન પર ક્લિક કરીને આ JSON પેસ્ટ કરો.
4. તમારા IBM watsonx ના `API Key` અને `Deployment ID` ને કનેક્ટરમાં અપડેટ કરો.
5. ફ્લોને **Deploy** કરો. તમારી સિસ્ટમ લાઈવ થઈ જશે!
